
Веб-застосунок для аналізу та візуалізації даних

Опис 
Даний веб-застосунок є частиною бакалаврської кваліфікаційної роботи Журби Анни та призначений для аналізу та візуалізації даних у веб-середовищі.

У сучасних умовах обсяги даних, що генеруються різними інформаційними системами, постійно зростають. Це створює необхідність у розробці простих та доступних інструментів для їх обробки та аналізу.

Існуючі програмні рішення часто є складними у використанні, потребують додаткового програмного забезпечення або мають обмеження у безкоштовних версіях. У зв’язку з цим актуальною є розробка веб-застосунку, який дозволяє виконувати базовий аналіз даних без необхідності спеціальної підготовки користувача.

Мета проєкту
Створення веб-застосунку для аналізу та візуалізації даних, який забезпечує:
завантаження та обробку CSV-файлів;
автоматичний розрахунок статистичних показників;
побудову інтерактивних графіків;
аналіз кореляцій між змінними;
відображення результатів у зручному візуальному форматі.
Основний функціонал

Веб-застосунок дозволяє:
завантажувати та обробляти CSV-файли;
автоматично визначати структуру датасету;
виконувати базові статистичні обчислення;
будувати інтерактивні графіки та діаграми;
формувати кореляційну матрицю;
відображати результати аналізу у вигляді інтерактивного дашборду.

Архітектура системи

Розробка системи виконувалась поетапно:
Аналіз предметної області та існуючих рішень.
Проєктування архітектури веб-застосунку.
Проєктування структури бази даних.
Реалізація клієнтської та серверної частини.
Тестування та перевірка працездатності на реальних наборах даних.
Режими роботи застосунку
Демонстраційний режим

Працює без підключення бази даних та надає доступ до:
базового інтерфейсу користувача;
перегляду основних сторінок системи.
Повнофункціональний режим

При підключенні MySQL активується повний функціонал:
реєстрація користувачів;
авторизація користувачів;
завантаження та збереження даних;
робота з результатами аналізу;
адміністративні можливості.

Інструкція запуску
1. Клонування репозиторію
git clone <URL_репозиторію>
2. Перехід до директорії проєкту
cd data_analytics_app
3. Встановлення залежностей
npm install
4. Запуск сервера
node server.js
5. Відкриття застосунку
Після успішного запуску сервер буде доступний за адресою:
http://localhost:3000
Налаштування бази даних

Для повноцінної роботи рекомендується використовувати phpMyAdmin та створити (або підключити) базу даних:
analytics_db

У цьому режимі стають доступними:
керування користувачами;
зміна ролей;
призначення адміністратора;
моніторинг даних;
збереження результатів аналізу;
повний функціонал системи.
