Web Application for Data Analysis and Visualization
Опис проєкту

Даний веб-застосунок є частиною бакалаврської кваліфікаційної роботи Журби Анни та призначений для аналізу та візуалізації даних у веб-середовищі.

У сучасних умовах обсяги даних, що генеруються різними інформаційними системами, постійно зростають. Це створює необхідність у розробці простих та доступних інструментів для їх обробки та аналізу. Існуючі програмні рішення часто є складними у використанні, потребують додаткового програмного забезпечення або мають обмеження у безкоштовних версіях. У зв’язку з цим актуальною є розробка веб-застосунку, який дозволяє виконувати базовий аналіз даних без необхідності спеціальної підготовки користувача.

Мета проєкту

Метою розробки є створення веб-застосунку для аналізу та візуалізації даних, який забезпечує завантаження та обробку CSV-файлів, автоматичний розрахунок статистичних показників, побудову інтерактивних графіків, аналіз кореляцій між змінними та відображення результатів у зручному візуальному форматі.

Основний функціонал

Веб-застосунок дозволяє завантажувати та обробляти CSV-файли, автоматично визначати структуру датасету, виконувати базові статистичні обчислення, будувати інтерактивні графіки та діаграми, формувати кореляційну матрицю, а також відображати результати аналізу у вигляді інтерактивного дашборду.

Використані технології

У процесі розробки використано сучасні веб-технології, зокрема HTML5, Tailwind CSS, JavaScript, Node.js, Express.js, систему керування базами даних MySQL, бібліотеку Chart.js для візуалізації даних, а також jsPDF і html2canvas для експорту результатів.

Архітектура системи

Розробка системи виконувалась поетапно. Спочатку було проведено аналіз предметної області та існуючих рішень. Далі здійснено проєктування архітектури веб-застосунку та структури бази даних. Наступним етапом стала реалізація клієнтської та серверної частини системи. Завершальним етапом було тестування та перевірка працездатності застосунку на реальних наборах даних.

Режими роботи застосунку

Веб-застосунок підтримує два режими роботи. У демонстраційному режимі система працює без підключення бази даних і надає доступ до базового інтерфейсу користувача та перегляду основних сторінок. У повнофункціональному режимі, при підключенні MySQL бази даних, активується повний функціонал системи, включаючи реєстрацію та авторизацію користувачів, завантаження та збереження даних, а також роботу з результатами аналізу.

Інструкція запуску

Для запуску застосунку необхідно завантажити проєкт з репозиторію GitHub за допомогою команди git clone. Після цього потрібно перейти у директорію проєкту та встановити всі залежності командою npm install. Далі запускається сервер командою node server.js. Після успішного запуску серверної частини застосунок буде доступний у браузері за адресою http://localhost:3000.

Примітка щодо бази даних

Для повноцінної роботи з базою даних рекомендується використовувати phpMyAdmin та підключити базу даних `analytics_db`. У такому режимі стають доступними всі можливості веб-застосунку, як адміністративні, так і клієнтські. Зокрема, з’являється можливість керування користувачами (наприклад, зміна ролей та призначення адміністратора), а також здійснення контролю та моніторингу даних у базі.
