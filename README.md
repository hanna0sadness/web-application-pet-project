ОПИС:
Веб-застосунок для аналізу та візуалізації даних є частиною моєї бакалаврської кваліфікаційної роботи та призначений для аналізу та візуалізації даних у веб-середовищі.


МЕТА- створення веб-застосунку для аналізу та візуалізації даних, який забезпечує:
1) завантаження та обробку CSV-файлів;
2) автоматичний розрахунок статистичних показників;
3) побудову інтерактивних графіків;
4) аналіз кореляцій між змінними;
5) відображення результатів у зручному візуальному форматі.


Веб-застосунок дозволяє:
1) завантажувати та обробляти CSV-файли;
2) автоматично визначати структуру датасету;
3) виконувати базові статистичні обчислення;
4) будувати інтерактивні графіки та діаграми;
5) формувати кореляційну матрицю;
6) відображати результати аналізу у вигляді інтерактивного дашборду.


Використані технології:
1) Frontend: HTML5, Tailwind CSS, JavaScript;
2) Backend: Node.js, Express.js;
3) База даних: MySQL;
4) Бібліотеки: Chart.js, jsPDF, html2canvas.


РЕЖИМИ РОБОТИ: 
1) Демонстраційний режим (Працює без підключення бази даних та надає доступ до базового інтерфейсу користувача та перегляду основних сторінок системи).

2) Повнофункціональний режим (При підключенні MySQL активується повний функціонал: реєстрація користувачів; авторизація користувачів; завантаження та збереження даних; робота з результатами аналізу; адміністративні можливості).


Інструкція запуску
1. Клонування репозиторію
git clone <https://github.com/hanna0sadness/web-application-pet-project>
2. Перехід до директорії проєкту
cd data_analytics_app
3. Встановлення залежностей
npm install
4. Запуск сервера
node server.js
5. Відкриття застосунку
Після успішного запуску сервер буде доступний за адресою:
http://localhost:3000


Налаштування бази даних MYSQL: для повноцінної роботи рекомендується використовувати phpMyAdmin та підключити базу даних analytics_db яка розміщена в даному репозиторії. Після підключення та налаштування бази даних стає доступними повний функціонал системи.
